# Teen Mental Health Prediction System

## Overview

This project predicts the risk of depression in teenagers using Machine Learning. The system analyzes lifestyle and mental health factors such as social media usage, sleep hours, stress level, anxiety level, addiction level, academic performance, and physical activity.

## Features

* Predicts depression risk in teenagers
* Displays probability percentage of depression risk
* User-friendly Streamlit web application
* Built using Random Forest Classifier
* Interactive input sliders and dropdowns

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Joblib

## Machine Learning Model

Algorithm: Random Forest Classifier

Accuracy: Approximately 98%

## Input Features

* Age
* Gender
* Daily Social Media Hours
* Platform Usage Level
* Sleep Hours
* Screen Time Before Sleep
* Academic Performance
* Physical Activity
* Social Interaction Level
* Stress Level
* Anxiety Level
* Addiction Level
* Mental Health Risk Score
* Sleep Quality
* Digital Wellbeing

## Output

* Low Risk of Depression
* Moderate Risk of Depression
* High Risk of Depression
* Depression Probability Percentage

## How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the Streamlit application:
   streamlit run app.py
